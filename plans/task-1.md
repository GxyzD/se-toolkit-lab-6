- принимает вопрос

- получает ответ

- Модель: qwen3-coder-plus
- Адрес API: http://10.93.26.116:12345/v1 (порт 12345 - замените на ваш)
- Ключ API: тот, который я указал в .env

## Структура программы
1. Прочитать настройки из файла .env.agent.secret
2. Взять вопрос из командной строки
3. Отправить запрос на API
4. Получить ответ
5. Вывести в формате JSON
